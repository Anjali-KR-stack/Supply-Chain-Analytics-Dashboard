# Supply Chain Analytics Dashboard

## Project Overview

This project analyzes supply chain operations to monitor shipment performance, delivery efficiency, carrier utilization, and plant productivity.

## Tools Used

- Microsoft Excel
- MySQL
- Power BI
- Power Query
- DAX

## Dataset Overview

The dataset contains:

- Order Details
- Carrier Information
- Plant Information
- Freight Rates
- Warehouse Capacities
- Product Information
- Customer Information

**Total Orders Analyzed:** 8,823

## Data Cleaning Process

- Removed duplicate records
- Standardized column names
- Validated data types
- Handled missing values
- Verified shipment records
- Imported cleaned data into MySQL

## Dashboard KPIs

- Total Orders
- Total Units Shipped
- Total Shipment Weight
- Average Transit Days
- On-Time Delivery %

## Key Insights

- PLANT03 handled the highest shipment volume.
- Carrier V444_0 processed the majority of shipments.
- DTP was the most frequently used service level.
- Average transit time was approximately 1.75 days.
- On-Time Delivery Rate exceeded 97%.

## Business Impact

The dashboard enables logistics teams to monitor operational performance, identify shipment bottlenecks, and improve delivery efficiency through data-driven decision-making.
