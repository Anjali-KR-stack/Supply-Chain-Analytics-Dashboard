# Supply Chain Analytics Dashboard

## Project Overview

This project focuses on analyzing supply chain operations using Excel, MySQL, and Power BI. The objective was to monitor shipment performance, carrier utilization, service level distribution, product movement, and transit efficiency through an interactive dashboard.

---

## Tools Used

- Microsoft Excel
- MySQL
- Power BI
- Power Query
- DAX

---

## Dataset Overview

The dataset contains supply chain information including:

- Orders
- Products
- Plants
- Carriers
- Freight Rates
- Warehouse Capacities
- Ports
- Customers

Total records analyzed: **8,823+ orders**

---

## Data Cleaning Process

- Removed duplicate records
- Renamed and standardized column headers
- Validated data types
- Checked for missing values
- Imported cleaned datasets into MySQL
- Performed data validation before dashboard creation

---

## SQL Analysis

Business analysis was performed using MySQL to identify:

- Total Orders
- Total Units Shipped
- Total Shipment Weight
- Plant-wise Shipment Volume
- Carrier Performance
- Service Level Distribution
- Product Shipment Analysis
- Transit Time Analysis

---

## Dashboard KPIs

- Total Orders: **9K**
- Total Units Shipped: **30M**
- Total Shipment Weight: **183.12K**
- Average Transit Days: **1.72**
- On-Time Delivery Rate: **98%**

---

## Key Insights

- More than **8,800 orders** were analyzed across the supply chain network.
- Approximately **30 million units** were shipped.
- Average shipment transit time was **1.72 days**.
- Carrier **V444_0** handled the highest number of shipments.
- **DTP** was the most frequently used service level, accounting for approximately **67%** of all shipments.
- Shipment performance maintained an on-time delivery rate of approximately **98%**.
